Can persistently store and retrieve objects, and notify listeners of changes to those objects

Parameters

T

The type of the objects to store, these must be serializable with Gson,

directory

The path to a directory in which data will be stored, will be created if it doesn't already exist

kc

The KClass associated with T. To avoid having to provide this use Shoebox<T>(directory)
